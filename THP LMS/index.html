Building an LMS with Legacy Hardware at Crooms Academy

Project Overview

In my role as a technology teacher at Crooms Academy of IT, I have embarked on an ambitious project to utilize outdated hardware for the creation of a comprehensive Learning Management System (LMS) for the Home School Partners. The project leverages two primary pieces of hardware: a Dell OptiPlex 3050 and a Dell R715 server. Below are the technical specifications that outline the capabilities of each system.

Dell OptiPlex 3050 Specifications:

Storage Configuration

  • 256GB NVMe SSD designated as the primary drive for the operating system
  • 120GB SATA SSD used for caching
  • 500GB 7200RPM SATA HDD for file storage
  • Secondary storage provided by a 2.5" 1TB 5400RPM HDD

Memory

  • 32GB of DDR4 RAM (2x16GB configuration)

Operating System

  • Ubuntu Server 20.04 LTS

Web Server

  • Options include Apache or Nginx

Database and Scripting

  • PHP scripting language with MariaDB or MySQL for database management

Caching

  • Redis or Memcached for data caching

Monitoring

  • Prometheus and Grafana for system monitoring

Backups

  • External USB HDD for backup solutions

Interface

  • LCD console display, keyboard, and mouse for direct interaction

Networking

  • Ethernet networking cables and an optional WiFi card for connectivity

DELL R715 Specifications:

Processors

  • Dual AMD Opteron 6168 12-core processors

Memory

  • 64GB of DDR3 ECC RAM (16x4GB configuration)

Storage

  • 6 x 300GB 10K SAS HDDs in a RAID 10 configuration for the database
  • 2 x 1TB 7200RPM SATA HDDs for additional backups and storage

Graphics and Network

  • AMD FirePro 2270 PCIe graphics card
  • Dual 1GbE NICs for network management

Power Supply

  • Dual 750W hot-swap power supply units

Remote Management

  • iDRAC7 Enterprise for system management

Operating System

  • Ubuntu Server 20.04 LTS

Web Server and Database

  • Web server options include Apache or Nginx
  • Database options include MySQL or PostgreSQL

Scripting and Applications

  • PHP for server-side scripting
  • Moodle as the LMS application

Monitoring

  • Implementation of monitoring tools such as Prometheus and Grafana

Project Significance

This integration of hardware and software is designed to yield a robust and reliable LMS platform suitable for the educational needs of the Home School Partners. Utilizing legacy hardware not only demonstrates resourcefulness and sustainability but also provides a cost-effective solution for digital learning initiatives. The selection and configuration of each component ensure that the system is capable of delivering a seamless online learning experience, emphasizing stability, performance, and scalability.

The project serves as a practical application of technology education, showcasing how theoretical knowledge can be translated into impactful real-world solutions. Through this initiative, I aim to foster an environment that not only imparts education but also encourages innovation and the meaningful repurposing of technology.
